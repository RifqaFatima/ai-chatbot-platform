# AI Chatbot Platform

A multi-tenant SaaS platform that lets users create AI-powered chatbots 
trained on their own documents and embed them on any website.

## What it does
- Users sign up and create chatbots
- Upload PDFs, docs, or text files as knowledge base
- Get an embeddable widget script for any website
- Visitors on their website can chat with the AI in real time

## Tech Stack
- Next.js 14 (App Router) + TypeScript
- PostgreSQL + Prisma
- NextAuth for authentication
- Google Gemini for AI
- Socket.IO for real-time chat
- Redis + BullMQ for queues
- AWS EC2 for deployment
